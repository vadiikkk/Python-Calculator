# Простейший калькулятор

## Описание задания

Необходимо реализовать простейший калькулятор, поддерживающий четыре арифметические операции:

| Метод       | Описание   |
|------------|-----------|
| `add`      | Сложение  |
| `subtract` | Вычитание |
| `multiply` | Умножение |
| `divide`   | Деление   |

## Что нужно сделать

Откройте файл `calculator.py` и реализуйте все четыре метода класса `Calculator`.

### Требования

- Все методы принимают два аргумента типа `float` и возвращают `float`.
- Метод `divide` должен выбрасывать `ValueError`, если делитель равен нулю.

## Запуск тестов

```bash
pytest -v
```
